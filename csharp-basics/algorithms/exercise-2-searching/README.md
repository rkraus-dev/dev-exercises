# Searching
Exercise practicing linear search and binary search algorithms.

## Task
Implement a `LinearSearch(int[], int)` method that works on unsorted arrays, and a `BinarySearch(int[], int)` method that works on sorted arrays by repeatedly halving the search range. Test both methods with values that exist and values that don't.

## Features
- `LinearSearch` scanning every element sequentially
- `BinarySearch` using a left/right pointer approach on a sorted array
- Test cases for found and not-found values on both methods

## Example Output
```
Linear search for: 20 | Found at Index: 5
Linear search for: 109 | Found at Index: -1

Binary search for: 57 | Found at Index: 7
Binary search for: 109 | Found at Index: -1
```
