# Class Basics

Exercise practicing class definition with fields and methods in C#.

## Task
Create a `Character` class with private fields, public setter methods, and a method that prints all field values. Create at least two independent objects, set different values on each, and print their info to demonstrate that each object holds its own state.

## Features
- `Character` class with private fields `name`, `level`, and `health`
- `SetName()`, `SetLevel()`, `SetHealth()` methods for controlled field access
- `PrintInfo()` method printing formatted character data
- Two independent `Character` objects with separate state

## Example Output
```
Name: Rudeus, Level: 27, Health: 100
Name: Elsa, Level: 20, Health: 100
```
