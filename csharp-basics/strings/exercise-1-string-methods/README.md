# String Methods
Exercise practicing common string methods in C#.

## Task
Take a messy input string with leading/trailing whitespace and mixed casing, clean it up using `Trim()` and `ToLower()`, check for a specific word using `Contains()`, replace a word using `Replace()`, and split the cleaned text into individual words using `Split()`.

## Features
- Text cleanup using `Trim()` and `ToLower()`
- `Contains()` used to check for a specific word in the cleaned text
- `Replace()` used to substitute a word in the cleaned text
- `Split()` used to break the text into individual words
- Word count and individual words printed using `foreach`

## Example Output
```
Does it contain the word "World"?: True
bonjour world, welcome to c#!
Length: 5
hello
world,
welcome
to
c#!
```
