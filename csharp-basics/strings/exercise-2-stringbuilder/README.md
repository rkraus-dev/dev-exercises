# StringBuilder
Exercise practicing StringBuilder for efficient string construction in C#.

## Task
Build a formatted shopping list using `StringBuilder`. Append each item in a loop, insert a heading at the beginning, replace one item, and print the final result along with its character length.

## Features
- `StringBuilder` used to build a numbered list from a `List<string>` in a loop
- `AppendLine()` used to add each item on its own line
- `Insert()` used to add a heading at the beginning
- `Replace()` used to substitute an item
- Final output printed via `ToString()`, along with `Length`

## Example Output
``` 
Shopping List:

Steak
Bananas
Pasta
Rice
Shrimps

Length: 62
```
