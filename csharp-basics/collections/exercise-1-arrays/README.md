# Arrays
Exercise practicing one-dimensional and multi-dimensional arrays in C#.

## Task
Create a one-dimensional array and print all values, then calculate and print the sum and the highest value using separate methods. Additionally, create a 3x3 two-dimensional array and print it row by row using nested loops.

## Features
- `CalculateSum(int[])` returning the sum of all array elements
- `FindMax(int[])` returning the highest value in the array
- 2D array printed row by row using `Console.Write` and `Console.WriteLine`

## Example Output

Biggest Value: 3002
