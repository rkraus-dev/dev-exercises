# List<T>
Exercise practicing List<T> operations in C#.

## Task
Build a small inventory management example using `List<string>`. Add items, remove an item, check whether a specific item exists using `Contains`, and print all remaining items along with the total count.

## Features
- Initial list populated with starting items
- `Add()` used to add new items
- `Remove()` used to remove an item by value
- `Contains()` used to check for a specific item
- `foreach` loop printing all remaining items
- `Count` printed to show the total number of items

## Example Output
```
Sword is in inventory: True
Shield
Sword
Boots
Cape
Total: 4
```
