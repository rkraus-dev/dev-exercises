# Character Sheet

Displays a character profile using multiple C# data types.

## Features
- Stores character data in typed variables
- Outputs a formatted character sheet to the console

## Example Output
```
=== Character Sheet ===
Name: Rudeus Greyrat
Level: 42
Mana: 9823.5
Gold: 1500,75 G
Is Reincarnated: True
``` 

## Exercise
C# Basics – Variables & Datatypes

## Topics Practiced
- int, double, decimal, bool, string
- Variable declaration and assignment
- decimal m-suffix
- Console.WriteLine
