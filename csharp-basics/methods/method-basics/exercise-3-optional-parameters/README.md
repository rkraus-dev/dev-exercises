# Optional Parameters
Exercise practicing optional parameters, default values, and named arguments in C#.

## Task
Create a `CreateCharacter` method with one required parameter and several optional parameters with default values. Call the method multiple times to demonstrate default values, positional arguments, and named arguments.

## Features
- `CreateCharacter(string, int, int, string)` with `name` as a required parameter
- Optional parameters `level`, `health`, and `characterClass` with default values
- Demonstrates positional argument calls
- Demonstrates named argument calls to skip middle parameters

## Example Output
```
Eris - Level 1 - HP 100 - Class: Adventurer
Rudeus - Level 10 - HP 100 - Class: Adventurer
Kinoas - Level 1 - HP 90 - Class: Adventurer
Hairo - Level 999 - HP 100 - Class: God
```
