# Greeting Flow
Basic exercise practicing method definition and method calls in C#.

## Task
Create a console program with three separate `void` methods (no parameters) that print a greeting, a divider, and a farewell message. Call all three methods from `Main` in a meaningful order.

## Features
- `PrintGreeting()` method printing a welcome message
- `PrintDivider()` method printing a visual separator
- `PrintFarewell()` method printing a farewell message

## Example Output
```
Hello, adventurer!
Safe travels!
```
