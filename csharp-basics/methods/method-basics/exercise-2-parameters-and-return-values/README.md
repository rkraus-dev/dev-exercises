# Parameters and Return Values
Exercise practicing method parameters, return values, and the difference between void and non-void methods.

## Task
Create methods that accept parameters and return calculated values: one method sums two numbers, one calculates the average of three numbers, and one prints a formatted result. Call the calculation methods from `Main`, store the results in variables, and print them.

## Features
- `AddNumbers(int, int)` returning the sum of two numbers
- `CalculateAverage(int, int, int)` returning the average as a `double`
- `PrintResult(string, double)` printing a formatted label and value
- Explicit cast to `double` to avoid integer division

## Example Output
```
Sum: 10
Average: 4
```
