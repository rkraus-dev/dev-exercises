# Level Check

Demonstrates comparison and logical operators through a series of character stat checks.

## Features
- Checks if level exceeds a threshold
- Checks if mana is within a range
- Combines conditions using AND, OR, and NOT operators

## Example Output
```
=== Level Check ===
Level:           75
Mana:            12000
Is Reincarnated: True

=== Checks ===
Level > 50:                      True
Mana between 5k-20k:             True
Reincarnated AND Lv > 50:        True
NOT Reincarnated OR Mana < 1000: False
```

## Exercise
C# Basics – Operators

## Topics Practiced
- Comparison operators (>, <)
- Logical operators (&&, ||, !)
- Combining multiple conditions
