# Combat Calculator

Simulates a combat scenario using arithmetic and logical operators.

## Features
- Calculates raw damage from attack power and mana multiplier
- Doubles enemy defense if the enemy is elite
- Determines if the attack results in a victory

## Example Output
```
=== Combat Calculator ===
Attack Power:    150
Mana Multiplier: 1,5
Defense:         100
Is Elite:        True

=== Results ===
Raw Damage:   225
Final Damage: 25
Victory:      True
```
