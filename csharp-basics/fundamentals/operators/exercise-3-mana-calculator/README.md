# Mana Calculator

Calculates total mana, spell costs, and remaining mana after casting using arithmetic and logical operators.

## Features
- Calculates total mana from base mana and level
- Applies a 25% prodigy discount on spell cost
- Computes remaining mana after three spells
- Checks if remaining mana is above half of total mana

## Example Output
```
=== Mana Calculator ===
Base Mana:  5000
Level:      42
Is Prodigy: True

=== Results ===
Total Mana:     11300
Cost per Spell: 1695
Remaining Mana: 6215
Above Half:     True
```
