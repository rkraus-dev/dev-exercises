# Conversion Chain

Demonstrates a chain of type conversions from string to double to int to decimal.

## Features
- Parses a string to double using double.Parse()
- Converts double to int using explicit cast
- Converts int to decimal using implicit conversion
- Outputs each conversion step

## Example Output
```
=== Conversion Chain ===
string  → double:  9823,7
double  → int:     9823
int     → decimal: 9823
```
