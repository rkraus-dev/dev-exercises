# Type Conversion Demo

Demonstrates the three main type conversion methods in C#.

## Features
- Converts a double to int using explicit cast
- Converts a string to int using Parse
- Converts a bool to int using Convert

## Example Output
```
=== Type Conversion Demo ===
[Cast]    double 9823,7 → int 9823
[Parse]   string "42" → int 42
[Convert] bool True → int 1
```
