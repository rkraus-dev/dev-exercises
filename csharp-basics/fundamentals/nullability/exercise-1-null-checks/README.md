# Null Checks

Demonstrates null handling in C# using various null operators.

## Features
- Uses ?? to provide fallback values for null strings and ints
- Uses ?. to safely access properties on nullable types
- Uses ??= to assign a value only if null

## Example Output
``` 
=== Null Checks ===
Guild:           No Guild
Rank:            Unranked
Name Length:     0
Guild after ??=: Adventurers Guild
``` 

## Exercise
C# Basics – Nullability & Null-Checks

## Topics Practiced
- Nullable reference types (string?)
- Nullable value types (int?)
- Null-coalescing operator (??)
- Null-conditional operator (?.)
- Null-coalescing assignment (??=)
