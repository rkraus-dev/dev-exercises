# Party Stats

Displays statistics for a party of three characters and calculates their averages.

## Features
- Stores data for three party members using typed variables
- Calculates average level and mana across the party
- Outputs a formatted party overview to the console

## Example Output
``` 
=== Party Stats ===
Member 1: Rudeus Greyrat     | Level: 42 | Mana: 9823.5
Member 2: Eris Boreas        | Level: 38 | Mana: 4200.0
Member 3: Roxy Migurdia      | Level: 55 | Mana: 15000.0

=== Party Average ===
Avg Level: 45
Avg Mana:  9674.5
```
