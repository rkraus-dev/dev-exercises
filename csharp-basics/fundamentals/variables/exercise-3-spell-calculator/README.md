# Spell Calculator

Calculates effective mana and spell costs for a caster, including a prodigy discount.

## Features
- Calculates effective mana using a base value and multiplier
- Computes mana consumption per spell
- Applies a 20% prodigy discount if the caster is a prodigy
- Outputs a formatted spell cost breakdown to the console

## Example Output
``` 
=== Spell Calculator ===
Caster:        Rudeus Greyrat
Base Mana:     10000
Multiplier:    1,75
Spell Count:   5
Is Prodigy:    True

=== Results ===
Effective Mana:       17500,00
Mana per Spell:       3500,00
Prodigy Discount:     -700,00
Final Cost per Spell: 2800,00
```
