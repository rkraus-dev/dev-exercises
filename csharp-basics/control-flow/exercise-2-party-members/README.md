# Party Members Announcer
A program that announces each party member joining using a foreach loop.

## Features
- Iterates over an array of party member names
- Outputs a join message for each member

## Example Output
```
Subaru has joined the party.
Tanya has joined the party.
``` 

## Exercise
C# Basics – Control Flow: foreach loop

## Topics Practiced
- foreach loop syntax
- Array iteration
- String interpolation
