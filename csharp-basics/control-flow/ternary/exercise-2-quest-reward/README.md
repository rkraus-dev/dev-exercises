# Quest Reward Resolver
A program that determines a quest reward based on its difficulty using a nested ternary operator.

## Features
- Evaluates quest difficulty with a nested ternary expression
- Handles a special case for maximum difficulty before checking the general threshold
- Outputs the corresponding reward

## Example Output
```
Reward: Common Potion
```

## Exercise
C# Basics – Control Flow: nested ternary operator

## Topics Practiced
- Nested ternary operator (`? :`)
- Condition ordering and evaluation priority
- String variable assignment
