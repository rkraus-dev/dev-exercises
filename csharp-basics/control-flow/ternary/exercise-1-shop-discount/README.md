# Shop Discount Calculator
A program that determines a customer's discount based on their membership status using the ternary operator.

## Features
- Evaluates membership status with a single-line conditional expression
- Outputs the corresponding discount percentage

## Example Output
```
Discount: 20%
```

## Exercise
C# Basics – Control Flow: ternary operator

## Topics Practiced
- Ternary operator (`? :`)
- Boolean conditions
- String variable assignment
