# Monster HP Tracker
A program that simulates a monster taking damage over repeated attacks using a while loop until it is defeated.

## Features
- Reduces monster HP by a fixed damage amount on each iteration
- Outputs the remaining HP after each attack
- Announces defeat once HP drops to zero or below

## Example Output
```
Monster HP: 38
Monster HP: 26
Monster HP: 14
Monster HP: 2
Monster HP: -10
The monster has been defeated!
``` 

## Exercise
C# Basics – Control Flow: while loop

- while loop syntax
- Loop condition evaluation
- Compound assignment operator (-=)
