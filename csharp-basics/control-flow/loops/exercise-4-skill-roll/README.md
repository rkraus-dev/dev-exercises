# Skill Roll Tracker
A program that simulates rolling skill points repeatedly until reaching a target value, using a do-while loop.

## Features
- Executes at least one roll before checking the loop condition
- Increments the roll value after each iteration
- Stops once the roll reaches the target threshold

## Example Output
```
Rolled: 3
Rolled: 4
Rolled: 5
``` 

## Exercise
C# Basics – Control Flow: do-while loop

## Topics Practiced
- do-while loop syntax
- Difference between while and do-while execution order
- Increment operator (++)
- Loop condition robustness (< vs !=)
