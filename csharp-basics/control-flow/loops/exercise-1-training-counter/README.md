# Training Counter
A program that outputs a training completion message for each day over a 5-day training period using a for loop.

## Features
- Iterates through a fixed range of days
- Outputs a status message for each training day

## Example Output
```
Day 1: Training session complete.
Day 5: Training session complete.
```
## Exercise
C# Basics – Control Flow: for loop

## Topics Practiced
- for loop syntax (initializer, condition, increment)
- Loop range control
- String interpolation
