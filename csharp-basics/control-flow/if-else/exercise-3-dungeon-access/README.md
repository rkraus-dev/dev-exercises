# Dungeon Access Checker
A program that determines whether an adventurer is granted access to a dungeon based on their status, level, and guild membership.

## Features
- Immediately blocks blacklisted adventurers regardless of other conditions
- Evaluates level and guild badge combinations for different access tiers
- Uses string interpolation to display the dungeon name dynamically

## Example Output
```
You may enter Abyss of Eternity with an escort only.
```
