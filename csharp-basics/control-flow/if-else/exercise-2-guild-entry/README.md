# Guild Entry Checker
A program that determines whether an adventurer is accepted into a guild based on their level and equipment.

## Features
- Evaluates level and weapon possession using multiple conditions
- Covers all four possible combinations of requirements
- Uses string interpolation to display the guild name dynamically

## Example Output
```
Your weapon is impressive, but you need to reach level 20.
```
