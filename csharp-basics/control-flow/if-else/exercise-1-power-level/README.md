# Power Level Classifier
A program that classifies a character's rank based on their power level using if / else if / else conditions.

## Features
- Evaluates a hardcoded power level value
- Outputs the corresponding rank and a short description
- Covers four rank tiers from F-Rank to Isekai God

## Example Output
```
Isekai God – no one can stop you.
```

## Exercise
C# Basics – Control Flow: if / else if / else
