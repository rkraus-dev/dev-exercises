# Rank Title Classifier
A program that outputs an adventurer's title based on their numeric rank using a classic switch statement.

## Features
- Maps numeric rank values to rank titles
- Handles unknown ranks with a default case

## Example Output
```
B-Rank – A reliable adventurer.
``` 

## Exercise
C# Basics – Control Flow: switch

## Topics Practiced
- Classic switch statement
- Case labels and break statements
- Default case handling
