# Element Power Resolver
A program that outputs an adventurer's elemental ability based on their assigned element using a switch expression.

## Features
- Maps element strings to elemental abilities
- Uses a modern switch expression to evaluate and return a result
- Handles unrecognized elements with a default case

## Example Output
``` 
You wield Inferno Blast – enemies fear your flames.
``` 

## Exercise
C# Basics – Control Flow: switch expression

## Topics Practiced
- Switch expression syntax
- Pattern matching on string values
- Default case handling with `_`
