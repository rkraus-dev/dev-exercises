# FizzBuzz
Classic FizzBuzz implementation using loops and conditional operators.

## Task
Iterate through numbers 1 to 100 and print each number. If the number is divisible by 3, print "Fizz". If divisible by 5, print "Buzz". If divisible by both 3 and 5, print "FizzBuzz".

## Approach
Used a for loop from 1 to 100. The FizzBuzz condition is checked first to avoid incorrect output when a number is divisible by both 3 and 5.

## Example Output
```
1
2
Fizz
4
Buzz
Fizz
7
8
Fizz
Buzz
11
Fizz
13
14
FizzBuzz
...
``` 

## Time Complexity
O(n) — single pass through the range.
